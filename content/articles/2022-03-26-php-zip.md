## 環境需求

- php5.2 以上版本
- 安裝 zip 庫

以 `Ubuntu` 為例：


```bash
sudo apt update
sudo apt install -y php7.4-zip
```

## 壓縮範例程式

```php
<?php
//new ZipArchive 物件
$zip = new ZipArchive;

$zip->open('new.zip', ZipArchive::CREATE);

```
